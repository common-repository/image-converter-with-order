== Description ==

Add custom image with woocommerce order, after upload your image, image will be convert in black and white and also in original format, and both image will be send in order and with email.


= Features =
- Attached image with order.
- Convert image color to black and white image and send with order for print.
- You can enable or disable plugin function in admin product page.
- Image will also send by email both for admin and user.
	
== Installation ==

1. Upload "Image Converter With Order" to the "/wp-content/plugins/" directory.

2. Activate the plugin through the "Plugins" menu in WordPress.


== Frequently Asked Questions ==

= How to setup upload button =
You just need to enable plugin and it will work and you will find option in product panel where you can active upload option in product.
